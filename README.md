# SignalDeck Energy Plugin

This repository contains the energy plugins for the SignalDeck framework:

* Goodwe inverter
* PV overview
* Zappi Wallbox combined with batter state
* Battery / Metersimulator



## Installation

```bash
pip install signaldeck-plugin-energy


Entwicklung:
 ```
 py -m pip install -e . --config-settings editable_mode=compat
 ```
